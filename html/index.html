Snowball Token

Let's get the ball rolling!

Snowball is a deflationary smart token that utilises the snowball effect whilst rewarding holders & punishing paper hands.

Deflationary Supply

An initial supply of 1,000,000,000,000,000 at launch will be decreased significantly over time to increase the value of each token. 50% of the supply has been burned pre launch and 2.5% of each transaction is burned and will never be retrieved thus decreasing supply.

Liquidity Pool

A percentage of each transaction gets locked into LP forever. This provides the token with an ever increasing price floor while ensuring deflation, making each token more valuable. Liquidity is essential for scalability and ensuring smooth transactions. LP fee is set to 2.5%.

Holder Rewards

Every token transaction redistributes a portion back to all token holders who can relax and watch their holdings grow therefore incentivising holders while punishing paper hands, making the token more valuable. Redistribution fee is set at 5%.
